Référence API

L'API Korvia est une interface REST propre sur HTTPS. Toutes les requêtes et réponses sont en JSON. Utilisez-la pour synchroniser des actifs, pousser des ordres depuis un autre système, ou tirer l'historique vers votre outil de BI.

URL de basehttps://api.korvia.ca/v1

Authentification

Authentifiez-vous avec un jeton bearer depuis Paramètres → Clés API. Gardez votre clé secrète; elle porte les permissions du rôle sous lequel elle a été créée.

curl
curl https://api.korvia.ca/v1/equipements \
  -H "Authorization: Bearer kv_live_8f2c…" \
  -H "Accept-Language: fr-CA"

Passez Accept-Language pour des libellés localisés (fr-CA ou en-CA).

Erreurs & limites

L'API utilise les codes HTTP standards. Les erreurs renvoient un corps JSON avec un code et un message lisible. Vous avez 1 000 requêtes par minute par clé.

CodeSignification
200 / 201Succès
401Jeton manquant ou invalide
403Le jeton n'a pas la permission
404Ressource introuvable
429Limite de débit dépassée

Équipements

GET/v1/equipements

Liste tous les équipements. Supporte la pagination avec ?page et ?par_page.

Réponse · 200
{
  "data": [
    {
      "id": "eqp_2418",
      "code": "PAT-2",
      "nom": "Presse à pâte",
      "etat": "en_panne",
      "site": "Plateau",
      "disponibilite": 0.87
    }
  ],
  "page": 1, "total": 142
}
GET/v1/equipements/{id}

Récupère un équipement avec son historique complet, ses pièces et documents.

Ordres de travail

POST/v1/ordres

Crée un ordre de travail. Seuls titre et equipement_id sont requis.

Corps de requête
{
  "titre": "Fuite hydraulique — presse à pâte",
  "equipement_id": "eqp_2418",
  "priorite": "urgent",
  "assigne_a": "usr_mathieu",
  "taches": ["Cadenasser", "Remplacer joint 41-227"]
}
GET/v1/ordres
PUT/v1/ordres/{id}
DEL/v1/ordres/{id}

Pièces

GET/v1/pieces
POST/v1/pieces/{id}/ajustement

Ajuste les niveaux de stock (réceptions, comptages, consommation). Renvoie la nouvelle quantité en main.

Webhooks

Abonnez-vous à des événements et Korvia enverra une charge JSON en POST à votre URL. Utile pour notifier Slack, mettre à jour un ERP, ou déclencher vos automatisations.

ÉvénementSe déclenche quand
ordre.creeUn ordre est créé
ordre.termineUn ordre est terminé
equipement.en_panneUn actif est marqué en panne
piece.sous_seuilUne pièce passe sous son point de commande
